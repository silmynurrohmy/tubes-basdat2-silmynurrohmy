API Documentation

Sistem Informasi Akademik ini dibangun menggunakan framework CodeIgniter 3 dan database MySQL. Sistem digunakan untuk mengelola data akademik yang meliputi mahasiswa, dosen, mata kuliah, kelas, dan nilai.

Fitur Utama

## Autentikasi Pengguna

Memungkinkan pengguna untuk login dan logout dari sistem.

## Manajemen Mahasiswa

Mengelola data mahasiswa meliputi tambah, ubah, lihat, dan hapus data.

## Manajemen Dosen

Mengelola data dosen meliputi tambah, ubah, lihat, dan hapus data.

## Manajemen Mata Kuliah

Mengelola data mata kuliah yang tersedia pada sistem.

## Manajemen Kelas

Mengelola data kelas yang digunakan dalam proses perkuliahan.

## Manajemen Nilai

Mengelola data nilai mahasiswa pada setiap mata kuliah.

## Dashboard

Menampilkan informasi ringkasan data akademik.

## Teknologi yang Digunakan

- PHP
- CodeIgniter 3
- MySQL
- HTML
- CSS
- JavaScript

## Database

Database yang digunakan adalah MySQL dengan beberapa tabel utama:

- mahasiswa
- dosen
- mata_kuliah
- kelas
- nilai
- user
